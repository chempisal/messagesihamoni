ការភ្ជាប់ទំនាក់ទំនងជាមួយ Google Sheets

របៀបដំឡើង និងភ្ជាប់ជាមួយ Google Sheets
  1. បង្កើត Google Sheet រួចចូលទៅកាន់ **Extensions > Apps Script**។
  2. ចម្លងកូដ Apps Script ដាក់ចូលក្នុង script editor រួចចុច **Save**។
  3. ចុច **Deploy > New Deployment** រួចជ្រើសរើសប្រភេទ **Web App**។
  4. កំណត់ Execute as: **Me** និង Who has access: **Anyone**។
  5. ចុច Deploy រួចចម្លង Web App URL មកដាក់ក្នុងប្រអប់ខាងលើ!
